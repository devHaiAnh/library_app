### Prequisites

#### Back-end: Python

- python
- flask
- psycopg2-binary
- flask-sqlalchemy
- Flask-Migrate

#### Front-end: Flutter

- smooth_star_rating
- font_awesome_flutter
- badges
- flutter_spinkit
- intl
- cached_network_image
- qr_flutter
- curved_navigation_bar
- bloc & flutter_bloc
- equatable
- http
- shared_preferences

- postgresql
